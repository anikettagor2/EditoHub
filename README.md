# EditoHub - Premium Video Editing Agency

A high-end, cinematic website for a professional video editing agency. Built with Next.js 15, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- **Premium Cinematic Design**: Dark theme, smooth gradients, and glassmorphism effects.
- **Smooth Animations**: Powered by Framer Motion for scroll reveals and micro-interactions.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop.
- **Modern Tech Stack**: Next.js App Router, React Server Components.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app`: App Router pages and layout.
- `src/components`: Reusable UI components and page sections.
- `src/lib`: Utility functions (cn).
- `src/app/globals.css`: Tailwind v4 configuration and global styles.

## Deployment

The project is ready to be deployed on Vercel.
