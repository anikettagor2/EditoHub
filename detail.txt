# EDITOHUB: UI/UX DESIGN SPECIFICATION & ARCHITECTURE
=====================================================

## 1. DESIGN PHILOSOPHY
-----------------------
"Enterprise-Grade Precision meets Agency-Level Aesthetics"
The Editohub UI is designed to feel like a high-performance command center. It utilizes a cinematic design language characterized by:
- Modern glassmorphism (backdrop blurs, subtle borders).
- Micro-interactions (hover translations, pulse animations).
- Semantic hierarchy (clear status differentiation through color and icons).
- Seamless transitions (Framer Motion page-fade and layout animations).

## 2. CORE TECH STACK
---------------------
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS v4 (with semantic CSS variables)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Theming:** next-themes (Light/Dark Mode Support)
- **Components:** shadcn/ui (Radix UI Adaptations)
- **Typography:** 
    - Heading: "Outfit" (tracking-tight, font-black)
    - Body: "Inter" (clean, highly readable)

## 3. DESIGN SYSTEM (SEMANTIC TOKENS)
-------------------------------------
The UI relies on a robust set of CSS variables that shift based on the active theme:

### Primary Palette (Electric Indigo)
- **Primary:** #4F46E5 (Light) / #6366F1 (Dark)
- **Accent:** #F3F4F6 (Light) / bg-white/[0.05] (Dark)

### Functional Colors
- **Success (Active):** Emerald-500
- **Warning (Idle/Draft):** Amber-500
- **Process (Review):** Purple-400
- **Destructive:** Red-500

### Surface & Depth
- **Background:** Primary application canvas.
- **Card:** Rounded-xl, enterprise-grade surface.
- **Muted/Popovers:** Secondary surfaces for dropdowns and navigation.
- **Border:** Soft transparency for layering (white/10 or border-border).

## 4. DASHBOARD ARCHITECTURE
----------------------------

### A. Client Dashboard (Request Portal)
- **Objective:** Minimal friction for project creators.
- **Key Features:**
    - Spending Analytics: Visual tracking of project value.
    - Project Timeline: Current status of all requests.
    - Quick Action: "New Request" button clearly accessible.

### B. Editor Dashboard (Production Suite)
- **Objective:** Task efficiency and response management.
- **Key Features:**
    - Task Matrix: Interactive table with "Accept/Reject" logic.
    - Fulfillment Timers: Real-time calculation of project age.
    - Production Status: Clear visual cues for 'todo' vs 'in review'.

### C. Admin Dashboard (Command Center)
- **Objective:** Full-spectrum visibility and control.
- **Key Features:**
    - System Stats: Real-time counter of active users and projects.
    - Template Preview: Live A4 rendering of HTML templates.
    - Bulk Management: Global search and filtering protocols.

### D. Project Manager Dashboard (Operations Matrix)
- **Objective:** Operational personnel linking (Node Linking).
- **Key Features:**
    - Pipeline Tracking: Monitoring unassigned project queues.
    - Resource Nodes: Editor availability and workload management (Max 5/editor logic).
    - Status Indicators: "Needs Node Link" vs "Active Production".

### E. Sales Dashboard (Acquisition Hub)
- **Objective:** Rapid client onboarding and revenue tracking.
- **Key Features:**
    - Entity Identity: Onboarding flows for new clients.
    - Revenue Ledger: Portfolio index and spending totals.
    - Auth Transmissions: Secure credential generation for new nodes.

## 5. GLOBAL UI COMPONENTS
--------------------------
- **Sidebar:** Role-based navigation with animated active states and persistent user profile.
- **Header:** Global search, system status heartbeat (Active Node Status), and secure protocol context.
- **Enterprise Cards:** Reusable container class with consistent hover lift and border transitions.
- **Status Indicators:** Micro-capsules with pulsing dots for real-time status representation.
- **Modals:** Centered, scale-animated overlays for critical actions like editor assignment.

## 6. USER EXPERIENCE FEATURES
------------------------------
- **Theme Awareness:** 100% semantic color coverage. No hardcoded hex values in component logic.
- **Device Responsiveness:** Fully fluid layouts from mobile (390px) to ultra-wide (1920px).
- **Ambient Glow:** Subtle radial gradients behind dashboard content for depth.
- **Accessibility:** Keyboard navigable menus and high-contrast text ratios.
